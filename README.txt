SPOTIFY TRENDS 2024

Desain Spotify Trends 2024 dibuat untuk menampilkan data musik secara interaktif dan mudah dieksplorasi. Tema dark dashboard dipilih agar visualisasi terlihat kontras dan nyaman dilihat, dengan warna-warna yang merepresentasikan identitas Spotify. Tata letak berbasis card grid memudahkan pengguna fokus pada satu grafik tanpa kehilangan konteks keseluruhan.

Setiap grafik memiliki fungsi spesifik: Bar chart menampilkan perbandingan 100 lagu terpopuler, donut chart menggambarkan proporsi lagu eksplisit, box plot menganalisis konsistensi performa artis, 3D scatter menunjukkan hubungan antar metrik TikTok, dan 3D surface memvisualisasikan korelasi lintas platform. Fitur interaktif seperti filter global, tooltip, ekspor grafik, serta scroll pagination ditambahkan untuk meningkatkan pengalaman pengguna.

Selama pengembangan, beberapa konsep awal disesuaikan karena keterbatasan teknis, namun sejumlah fitur baru seperti loading overlay dan desain responsif ditambahkan untuk memastikan performa dan tampilan web tetap optimal di berbagai perangkat.

Teknologi yang Digunakan:
- HTML5 — Struktur halaman web
- CSS3 — Desain visual dan tema gelap
- JavaScript (ES6) — Logika interaktif dan pengelolaan data
- D3.js — Visualisasi data 2D (Bar, Donut, Box Plot)
- Plotly.js — Visualisasi data 3D (Scatter, Surface)
- GitHub Pages — Deployment web publik
- Dataset: spotify_songs_2024.csv

Cara Menjalankan:
1. Unduh seluruh file proyek dari repositori.
2. Pastikan file spotify_songs_2024.csv berada di direktori utama.
3. Jalankan proyek menggunakan Live Server di Visual Studio Code atau buka index.html langsung di browser.
4. Untuk versi online, buka halaman GitHub Pages dari repository ini.
